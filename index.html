Welcome, Selamat Datang dan Kon'nichiwa


Bisa dibilang ini blog pertama saya langsung dari scratch ( dari awal bikinnya ). Mungkin tulisan ini fokus mengenai pengalaman saya pertama kali belajar mengenai IT khususnya membuat website. Dahulu pertama kali saya membuat website pada saat kuliah semester 4 diminta ama organisasi kampus untuk membuat website dengan tujuan untuk mengirimkan undangan untuk acara Musyawarah Nasional FoSSEI ( Forum Silaturahmi Ekonomi Islam). Ya tujuannya tak lain dan tak bukan untuk penghematan dana karena untuk memprint surat + mengirimkan ke semua universitas yang berada di Indonesia membutuhkan dana yang cukup lumayan pada saat itu. Awalnya saya dengan sigap mengatakan sanggup karena di otak saya “ ah palingan bikin wordpress gitu “ ternyata senior-senior inginya website yang berdiri sendiri . “Waduh mampus gue “dalam hati. Akhirnya dengan mau tidak mau saya mulai mencoba membuat website tersebut.


Saat ini enak apa - apa di google + youtube selesai ada stack overflow dan lainnya, pada saat thn 2006 dengan internet yang cukup sedih saat itu 512kbps ditambah latar belakang pendidikan saya dari Akuntansi tidak ada sama sekali hubungan ama IT saya mulai mencari informasi cara membuat website dan semua sumber - sumber itu saya dapat banyak dari blog dan forum dan yang kebanyakan berbahasa inggris, beruntung saat itu saya part time sebagai OP warnet di daerah kampus serta bahasa inggris sudah lumayan karena sejak kelas 6 SD udah disuruh les ama Ortu . Dari sumber - sumber itu saya tampung beberapa opsi untuk membuat website, yang pertama membuat dari awal alias mesti belajar semua dan waktu yang diberikan tidak memungkinkan , yang kedua menggunakan wordpress, joomla , drupal. Setelah research baca sana sini akhirnya pilihan saya jatuh ke Joomla karena terdapat banyak fitur yang cukup lengkap ( walau terkadang bingung banyak plugin fungsinya sama ).


Pada awalnya saya di langsung menginstall CMS Joomla pada hosting dan mulai membuat ancang - ancang apa saja yang saya akan bikin. Jelang seminggu saya merasa stuck karena kecepatan loading website dan kadang - kadang masih ada cache yang ada sehingga setelah saya melakukan penginstalan module , component dan plugin, dan sering kali setelah di install merusak tampilan websitenya sehingga membuat saya frustasi, setelah baca - baca di internet saya baru ngeh ternyata kita bisa install joomla di PC sendiri dan semua component, plugin bisa kita test terlebih dahulu sebelum ke live server menggunakan FTP. Akhirnya saya install XAMPP yang merupakan web server populer di masa itu dan mulai perjalanan mengotak atik Joomla berserta semua component dan pluginnya.


Hal yang pertama yang saya coba ada modul thema karena saya ingin thema nya rada hijau - hijau gitu keislaman mulailah cari - cari tema yang cocok dari gratis ampe yang semi bayar ( mereka minta letak iklan website mereka di web kita ), seingat saya butuh sekitar seminggu untuk mencari yang pas. Dan akhirnya saya mulai ngotak ngatik kode HTML dan ingat banget waktu itu masih pake adobe dreamweaver (asli VSC sekarang itu blessing banget thx Microsoft). Terus cari download buku - buku gratis belajar dreamweaver di situ yang mulai mengenal CSS (yang tetap sesuatu yang simple kadang rumit dipelajari sampai saat ini). Dari situ yang mulai mengganti warna background, repeat gambar, saat itu languange yang cukup populer adalah PHP, sehingga mulai saya belajar php hanya untuk sekedar bisa merubah beberapa fungsi yang ada untuk keperluan saya.


Setelah 3 minggu mengotak - atik CMS Joomla tampilannya dan logo nya sudah sesuai dengan apa yang saya rencanakan, dan hampir semua fungsi dan tujuan dari website itu sudah bisa berjalan dengan baik itu saya bolak balik edit dari pc ke hostingnya beberapa kali, dan untuk trial and error di PC dan final version saya implementasikan ke hosting. Ada kejadian yang menarik pada saat itu ada plugin dari Joomla yang bisa bikin content surat menjadi PDF dan bisa mengedit beberapa variabel seperti nama dan tanda tangan di situ saya mencoba untuk mengedit JS nya agar bisa mengedit variabel penerima sehingga saya ingin semua orang yang daftar menjadi penerima undangan tersebut dan nama di undangan sudah berubah sesuai dengan nama Universitas yang mendaftar, cukup lama juga untuk belajar mengenai itu memakan waktu hampir 1.5 minggu


Setelah websitenya live running saya minta tolong ke senior - senior untuk mencoba daftar website tersebut agar semua kesalahan proses dan bug bisa muncul dan dapat teratasi, dan emang iya muncul beberapa bug yang cukup fatal salah satunya adalah orang random mana aja bisa daftar dan mengaku - ngaku dari universitas mana sehingga waktu itu karena belum ada otentifikasi melalui no hp kita melakukannya secara manual sehingga saya cek di sheet excel benar tidak orangnya nah bagian manual itu saya tugaskan ke anggota - anggota organisasi lain. ( BTW karena rame yang mengakses website nya sempat down wkwkkw padahal hanya untuk hemat biaya doang ). Websitenya cukup membikin mahasiswa - mahasiswa Universitas lain berdecak kagum karena waktu itu KaSEI UNRI udah secanggih gitu orangnya ( denger gitu mayan besar kepala saya wkwkwk ) .


Setelah acara itu websitenya cukup banyak improvement, ya karena joomla banyak plugin, component dan modul yang kaya. Salah satu improvement besar adalah forum dan penulisan karya tulis. Sayang nya website ini hanya bertahan sekitar 3 tahun karena setelah saya tidak di organisasi dan lulus website itu tidak ada yang mengurus dan hostingnya tidak diperpanjang karena biaya ( maklum mahasiswa ). Setelah lulus saya sempat membuat web aplikasi surat menggunakan PHP local server untuk kantor di BSM pada saat bekerja disana (sempat dipakai sekitar 6 thn sampai akhirnya BSM membuat sendiri aplikasi internalnya ). Dan pada tahun 2020 saya sempat pernah mengikuti kursus online DICODING mengenai data dan tahun ini Id camp membuka peluang untuk belajar kembali tentang front end sehingga saya berterima kasih sekali atas kesempatan yang diberikan oleh Id camp dan dicoding. Semoga tulisan ini bisa menjadi salah satu nilai + bintang 5 submission saya ( walaupun color scheme ama bentuknya berantakan tolong di approve dungz :) )
