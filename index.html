Welcome, Selamat Datang dan Kon'nichiwa


Bisa dibilang ini blog pertama saya langsung dari scratch ( dari awal bikinnya ). Mungkin tulisan ini fokus mengenai pengalaman saya pertama kali belajar mengenai IT khususnya membuat website. Dahulu pertama kali saya membuat website pada saat kuliah semester 4 diminta ama organisasi kampus untuk membuat website dengan tujuan untuk mengirimkan undangan untuk acara Musyawarah Nasional FoSSEI ( Forum Silaturahmi Ekonomi Islam). Ya tujuannya tak lain dan tak bukan untuk penghematan dana karena untuk memprint surat + mengirimkan ke semua universitas yang berada di Indonesia membutuhkan dana yang cukup lumayan pada saat itu. Awalnya saya dengan sigap mengatakan sanggup karena di otak saya “ ah palingan bikin wordpress gitu “ ternyata senior-senior inginya website yang berdiri sendiri . “Waduh mampus gue “dalam hati. Akhirnya dengan mau tidak mau saya mulai mencoba membuat website tersebut.


Saat ini enak apa - apa di google + youtube selesai ada stack overflow dan lainnya, pada saat thn 2006 dengan internet yang cukup sedih saat itu 512kbps ditambah latar belakang pendidikan saya dari Akuntansi tidak ada sama sekali hubungan ama IT saya mulai mencari informasi cara membuat website dan semua sumber - sumber itu saya dapat banyak dari blog dan forum dan yang kebanyakan berbahasa inggris, beruntung saat itu saya part time sebagai OP warnet di daerah kampus serta bahasa inggris sudah lumayan karena sejak kelas 6 SD udah disuruh les ama Ortu . Dari sumber - sumber itu saya tampung beberapa opsi untuk membuat website, yang pertama membuat dari awal alias mesti belajar semua dan waktu yang diberikan tidak memungkinkan , yang kedua menggunakan wordpress, joomla , drupal. Setelah research baca sana sini akhirnya pilihan saya jatuh ke Joomla karena terdapat banyak fitur yang cukup lengkap ( walau terkadang bingung banyak plugin fungsinya sama ).


Pada awalnya saya di langsung menginstall CMS Joomla pada hosting dan mulai membuat ancang - ancang apa saja yang saya akan bikin. Jelang seminggu saya merasa stuck karena kecepatan loading website dan kadang - kadang masih ada cache yang ada sehingga setelah saya melakukan penginstalan module , component dan plugin, dan sering kali setelah di install merusak tampilan websitenya sehingga membuat saya frustasi, setelah baca - baca di internet saya baru ngeh ternyata kita bisa install joomla di PC sendiri dan semua component, plugin bisa kita test terlebih dahulu sebelum ke live server menggunakan FTP. Akhirnya saya install XAMPP yang merupakan web server populer di masa itu dan mulai perjalanan mengotak atik Joomla berserta semua component dan pluginnya.
